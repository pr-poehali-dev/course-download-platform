## Как это работает

### Для статических серверов (Apache/Nginx)
1. Пользователь заходит на `/non-existent-page`
2. Сервер не находит файл
3. Возвращает `404.html` со статусом **404 Not Found** ✅

### Для SPA (React Router)
1. Пользователь заходит на `/non-existent-page`
2. React Router обрабатывает маршрут
3. Показывает `NotFoundPage` компонент
4. Meta-тег указывает статус 404 (для SEO)
